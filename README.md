# Heart Disease Prediction Website ❤️

## Description 😀
- Heart Disease Prediction Website is a web-based application that predicts the risk of heart disease using user health inputs. It provides quick and accurate results through a simple and user-friendly interface.
- This website is built using the Flask web framework. For the front-end flask, Bootstrap is used & Mongodb is used as Database.
- Used RandomForestClassifier Machine Learning algorithm for model building which gives 90.16% accuracy.

## Packages Used 👀
- Flask
- Flask-PyMongo
- Numpy
- Pickle
   
## Tools Used 🛠
- Frontend: HTML, CSS, JavaScript
- Backend: Python(Flask)
- Database: MongoDB
